## Overview

React Ink brings React's component model to the terminal. Instead of rendering
to the DOM, Ink renders to stdout using a custom React reconciler backed by Yoga
layout engine (the same Flexbox implementation used by React Native). Build
interactive CLI tools with components like `<Box>` for layout and `<Text>` for
styled output, handle keyboard input with `useInput`, and manage focus with
`useFocus` - all using familiar React patterns including hooks, state, effects,
Suspense, and concurrent rendering.
